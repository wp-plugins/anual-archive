== Description ==

Annual Archive widget extends default WordPress Archive widget by allowing either monthly or annually archives to be displayed.  The Annual Archive widget - It displays archives by year.  Yup, that's pretty much it.

== Installation ==

1. Old-school: upload the annual-archive folder to the `/wp-content/plugins/` directory via FTP.  Hipster: Ironically add annual-archive via the WordPress Plugins menu.
1. Activate the Plugin
1. Test that the this plugin meets your demanding needs.
1. Rate the plugin and verify if it works at wordpress.org.
1. Leave a comment regarding bugs, feature request, cocktail recipes at http://www.twinpictures.de/anual-archive-widget/
